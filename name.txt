# EXPERIMENT 4 — Jenkins Installation and Setup

---


```text

```

Ignore and continue.

---

# Install Java (Required for Jenkins)

```bash

```

```bash

```

# Add Jenkins Repository Key

```bash
curl -fsSL https://pkg.jenkins.io/debian-stable/jenkins.io-2023.key | sudo tee \
/usr/share/keyrings/jenkins-keyring.asc > /dev/null
```

---

# Add Jenkins Repository

```bash
echo deb [signed-by=/usr/share/keyrings/jenkins-keyring.asc] \
https://pkg.jenkins.io/debian-stable binary/ | sudo tee \
/etc/apt/sources.list.d/jenkins.list > /dev/null
```

```bash

```



## Error: Port 8080 Already in Use

Check process using port:

```bash

```

Kill process:

```bash
sudo kill -9 <PID>
```

Restart Jenkins:

```bash
sudo systemctl restart jenkins
```

---

## Error: Permission Denied

Run commands using sudo:

```bash
sudo systemctl start jenkins
```

```java
class ReverseNumber
{
public static void main(String args[])
{
int n=Integer.parseInt(args[0]);

int rev=0;
int r;

while(n>0)

rev=(rev*10)+r;
```

```html
var prev,oper;

function num(s)
{

var sum=(n*10)+s;
document.f1.t1.value=sum;
}

function op(k)
{
oper=k;
prev=parseInt(document.f1.t1.value);
document.f1.t1.value=0;
}

function mod()
{
a=parseInt(document.f1.t1.value);

function eq()
{
curr=parseInt(document.f1.t1.value);

switch(oper)
{
case 1: var result=prev+curr;break;
case 2: var result=prev-curr;break;
case 3: var result=prev*curr;break;
case 4: var result=prev/curr;break;
case 5: var result=prev%curr;break;
}
```

## ERROR: docker command not found

Install Docker first.

---

## ERROR: kubectl command not found

Install Kubernetes tools:

```bash
snap install kubectl --classic
```

---

## ERROR: ImagePullBackOff

Use:

```yaml
imagePullPolicy: Never
```

inside deployment YAML.

## ERROR: ContainerCreating

Check Docker service:

```bash
systemctl status docker
```

Start Docker:

```bash
systemctl start docker
```

---

## ERROR: Git push rejected

Pull first:

```bash
git pull origin main --allow-unrelated-histories
```

Then push again.

## ERROR: Permission denied while pushing Docker image

Login again:

```bash
docker login -u rashmitaa21
```
